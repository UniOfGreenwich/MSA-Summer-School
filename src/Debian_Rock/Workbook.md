
## Updating the workbook

The workbook is hosted on GitHub in the university of Greenwich's organisational page. Therefore accessible any where with an internet connection. However, when we move to the IoT Network we will lose access, so the workbook is stored locally on this Rock C+4, when an updated is needed we can connect to the internet by disconnecting for a moment and run a command that updates the workbook.  

```sh
$ update-workbook
```

Remember to refersh you webpage to see reload the change.
